AI-Based Dermatological Diagnosis (Data Analysis Version)

Project Overview:
This project demonstrates a simple data analysis and machine learning workflow
for preliminary skin disease classification.

Steps Included:
1. Data loading
2. Exploratory data analysis
3. Data preprocessing
4. Machine learning model training (Random Forest)
5. Model evaluation and visualization

Dataset:
A small synthetic dataset is included for demonstration.

How to Run:
1. Install dependencies:
   pip install -r requirements.txt

2. Run analysis:
   python analysis.py
